Welcome to image recommender demo app! Like images

In the 'Random Images' section of the navigation bar you can access all the images available in the system. There you can select some images you like and proceede to the next step.

Inspected choices

In the 'Liked Images' section of the navigation bar you can access all the images the user has marked as favorites. Based on those images the system can find similar items which can be viewed in the final section.

Observe recommendations

In the 'Recommended Images' section of the navigation bar you can inspect the recommendations.
